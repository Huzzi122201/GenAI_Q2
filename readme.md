# 🎨 Anime Sketch Colorization with pix2pix GAN

Transform your sketches into vibrant anime-style colored images using a trained pix2pix GAN model.

## ✨ Features

- 🖼️ Upload any sketch or image
- 🎨 Real-time AI-powered colorization
- ✏️ Draw sketches directly in the app
- 📸 Convert photos to sketches
- 💾 Download generated results
- 🚀 GPU acceleration support

## 📋 Requirements

- Python 3.8+
- CUDA-capable GPU (optional, for faster inference)
